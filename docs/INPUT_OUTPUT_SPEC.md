# 入力・出力・境界値仕様

## 共通

| 項目 | 入力 | 正常時の出力 | 境界値・異常時 |
| --- | --- | --- | --- |
| 文字入力 | 会社名、現場名、作業内容、備考など | 前後空白と制御文字を除去して保存・表示 | PDF/API用は通常500文字、長文は2000文字まで |
| HTML出力 | 帳票プレビューの文字 | HTMLエスケープ後に表示 | `<script>` などは文字として表示し、実行しない |
| 画像ファイル | 領収書、資格証、車両書類、日報写真 | Data URLとして端末内保存 | 画像以外、0バイト、8MB超は保存しない |
| PDF API | title, rows, note | PDFを返す | 64KB超は413、JSON不正は400 |

## 予定追加

| 入力 | 正常時の結果 | 境界値 |
| --- | --- | --- |
| 日付 | カレンダーの該当日に予定を表示 | 空欄なら今日 |
| 現場名 | 予定名、日報、請求書へ反映 | 空欄なら「現場予定」 |
| 請求先会社名 | 請求書の宛先へ反映 | 空欄でも予定登録は可能 |
| 作業内容 | 予定一覧、日報初期値、請求書明細へ反映 | 必須 |
| 人工数 | 請求書の数量へ反映 | 空欄なら1 |
| 人工単価 | 請求書の単価へ反映 | 空欄なら現場単価、なければ25,000円 |

## 日報記入

| 入力 | 正常時の結果 | 境界値 |
| --- | --- | --- |
| 日付 | その日の日報として保存 | 同じ日付は上書き |
| 作業員 | 日報とカレンダーに表示 | 空欄なら未入力表示 |
| 今日やったこと | 日報本文、請求書作業内容へ反映 | 空欄でも保存可能 |
| 写真 | 最大12枚まで日報に保持 | 画像以外、8MB超は拒否 |

## 領収書保存

| 入力 | 正常時の結果 | 境界値 |
| --- | --- | --- |
| 領収書画像 | Storageに画像を保存し、手入力した項目と紐づける | 画像以外、0バイト、8MB超は保存しない |
| 保存項目 | 日付、金額、支払先、用途、メモ、現場、状態を保存 | 担当者と会社名はプロフィールから自動で扱う |
| 日付 | `YYYY-MM-DD` 形式で保存 | 不正な日付は空欄にしてフォームで補正 |
| 金額 | 手入力した金額を保存 | 空欄なら0円として扱う |
| 編集・削除 | 一覧から選択して同じIDで更新、不要なものは削除 | 削除は `app_id` 指定で実行する |

## 請求書・見積書

| 入力 | 正常時の結果 | 境界値 |
| --- | --- | --- |
| 番号 | 請求番号/見積Noとして帳票に表示 | 空欄なら日付ベースで自動作成 |
| 発行日 | 請求日/見積作成日として表示 | 空欄なら今日 |
| 件名 | 帳票の件名欄へ表示 | 空欄なら現場名または工事一式 |
| 支払条件 | 帳票の支払条件欄へ表示 | 空欄なら `-` |
| 備考 | 帳票の備考欄へ表示 | 2000文字まで |
| 金額 | 小計、消費税、税込合計を計算 | 税率0%、8%、10%に対応 |

## PDF帳票

| 入力 | 正常時の結果 | 境界値 |
| --- | --- | --- |
| 請求書データ | 請求書プレビューを開き、印刷・PDF保存できる | 危険なHTMLは文字として表示 |
| 見積書データ | 見積書プレビューを開き、印刷・PDF保存できる | 危険なHTMLは文字として表示 |
| 領収書一覧 | 領収書一覧プレビューを開き、印刷・PDF保存できる | 0件なら対象データなし表示 |
| 発行者情報 | プロフィール情報を帳票へ反映 | 未入力なら空欄または発行者未設定 |
| 帳票プレビュー | `/print` で共通表示 | ブラウザの印刷機能からPDF保存 |

## セキュリティ期待値

- 帳票に危険なHTMLを入力しても実行されない。
- 端末保存アカウントのパスワードはハッシュ化して保存する。
- 外部からPDF APIへ大きすぎるデータを送っても処理しない。
- アプリ全体に基本セキュリティヘッダーを付ける。
